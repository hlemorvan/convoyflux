== Architecture du store Redux

....
 store/
 ├── vehiclesSlice.ts
 │    state: Map<vehicleId, VehicleState>
 │    VehicleState: { current: Telemetry, history: Telemetry[50] }
 │    actions: upsertVehicle(Telemetry), selectVehicle(id|null)
 │
 ├── connectionSlice.ts
 │    state: 'connecting' | 'connected' | 'reconnecting' | 'error'
 │    actions: setStatus(ConnectionStatus)
 │
 └── filtersSlice.ts
      state: { searchId, region, minSpeed }
      actions: setSearchId, setRegion, setMinSpeed
....

== Flux de données

....
 EventSource /api/stream
       │ event: telemetry
       │ data: {...}
       ▼
 useTelemetryStream (hook)
       │ dispatch(upsertVehicle(telemetry))
       │ dispatch(setStatus('connected'))
       ▼
 vehiclesSlice reducer
       │ Map<id, VehicleState> mis à jour
       ▼
 selectors (memoïsés via createSelector)
       │ filteredVehicles — filtre ID + région + vitesse min
       ▼
 FleetMap ────► marqueurs MapLibre mis à jour
 VehiclePanel ► détail du véhicule sélectionné
....

== Composants principaux

=== FleetMap

image::map-live.png[Carte en direct,align=center]

// SVG inline — pas de fichier externe, taille fixe 26×26
const el = document.createElement('div')
el.innerHTML = `

== Hook useTelemetryStream

[source,typescript]
----
// EventSource avec reconnexion exponentielle (2s → 4s → … → 30s max)
function useTelemetryStream() {
  const dispatch = useDispatch()

  useEffect(() => {
    let attempt = 0
    let es: EventSource
    let timer: ReturnType<typeof setTimeout>

    function connect() {
      dispatch(setStatus(attempt === 0 ? 'connecting' : 'reconnecting'))
      es = new EventSource('/api/stream')

      es.addEventListener('telemetry', (e) => {
        const t = JSON.parse(e.data)
        dispatch(upsertVehicle(t))
        dispatch(setStatus('connected'))
        attempt = 0
      })

      es.onerror = () => {
        es.close()
        dispatch(setStatus('error'))
        const delay = Math.min(2 ** attempt * 1000, 30_000)
        attempt++
        timer = setTimeout(connect, delay)
      }
    }

    connect()
    return () => { es?.close(); clearTimeout(timer) }
  }, [dispatch])
}
----

== Thèmes clair / sombre

La préférence système est détectée au premier rendu via
`window.matchMedia('(prefers-color-scheme: dark)')`.
Un bouton bascule ajoute / retire la classe `dark` sur `<html>`.
MapLibre recharge le style de tuiles correspondant.
