=== Application — TelemetryService

[source,java]
----
@ExtendWith(MockitoExtension.class)
class TelemetryServiceTest {
    @Mock TelemetryRepository repo;
    @Mock TelemetryBroadcast  broadcast;

    @Test void ingest_savesAndPublishes() {
        var svc = new TelemetryService(repo, broadcast);
        var t   = telemetry();
        when(repo.save(t)).thenReturn(Mono.empty());

        svc.ingest(t).block();

        verify(repo).save(t);
        verify(broadcast).publish(t);
    }
}
----

== Tests frontend (Vitest)

=== Reducer Redux

[source,typescript]
----
it('upsertVehicle adds to state', () => {
  const t = telemetry('v-001')
  const s = reducer(undefined, upsertVehicle(t))
  expect(s.entities['v-001'].current.speed).toBe(t.speed)
})

it('history keeps at most 50 points', () => {
  let s = initialState
  for (let i = 0; i < 55; i++)
    s = reducer(s, upsertVehicle({ ...telemetry('v-001'), speed: i }))
  expect(s.entities['v-001'].history).toHaveLength(50)
})
----

=== Mocking SSE dans Playwright

[source,typescript]
----
await page.route('/api/stream', async route => {
  const body = [
    'event: telemetry\n',
    `data: ${JSON.stringify(payload)}\n\n`,
  ].join('')
  await route.fulfill({
    status: 200,
    headers: { 'Content-Type': 'text/event-stream' },
    body,
  })
})
----

NOTE: `route.fulfill()` ferme la connexion après envoi → `EventSource` repasse
en état d'erreur immédiatement. Ne pas attendre `✓ Connecté` — attendre
l'apparition des marqueurs à la place.

== Référence API REST

=== GET /api/stream

[cols="1,3",options="header"]
|===
|Paramètre|Description
|`Accept`|`text/event-stream` (obligatoire)
|===

Format de réponse :

[source]
----
event: telemetry
data: {
  "vehicleId": "v-042",
  "region":    "ile-de-france",
  "lat":       48.8566,
  "lng":       2.3522,
  "speed":     45.2,
  "heading":   127.0,
  "ts":        "2026-06-30T10:00:00Z"
}
----

=== GET /api/vehicles/{vehicleId}/history

[cols="1,1,2",options="header"]
|===
|Paramètre|Type|Description
|`vehicleId`|path|Identifiant du véhicule
|`limit`|query (défaut 50)|Nombre de points à retourner
|===

Réponse : `application/json` — tableau de `TelemetryDto` triés par `ts` DESC.
